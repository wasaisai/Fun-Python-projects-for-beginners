# 项目编码规范

本项目是一个 Python 初学者练习项目，包含各种小工具和练习代码。

## 编码格式

### 基本规范
- **编码**：UTF-8
- **缩进**：使用 4 个空格（不使用 Tab）
- **行长度**：建议不超过 100 字符
- **文件编码**：所有 Python 文件使用 `.py` 扩展名

### 命名规范
- **文件名**：使用小写字母和下划线，如 `is_list_mini.py`、`reverse_str.py`
- **函数名**：使用小写字母和下划线，如 `get_input()`、`calc_min()`、`is_valid()`
- **变量名**：使用小写字母和下划线，如 `input_list`、`mini_num`、`num_list`
- **常量名**：使用大写字母和下划线（如需要）

### 代码风格
- 函数之间使用 2 个空行分隔
- 函数内部逻辑块之间使用 1 个空行分隔
- 运算符两侧添加空格：`a = b + c`
- 逗号后面添加空格：`[1, 2, 3]`
- 避免行尾空格

## 函数文档规范

### 必须为每个函数添加文档字符串（docstring）

使用中文编写函数说明，格式如下：

```python
def function_name(param1, param2):
    """
    函数功能的简短描述

    参数:
        param1: 参数1的说明
        param2: 参数2的说明

    返回:
        返回值的说明
    """
    # 函数实现
    pass
```

### 示例

```python
def calc_min(input_list):
    """
    计算列表中的最小值

    参数:
        input_list: 包含数字字符串的列表

    返回:
        float: 列表中的最小数字
    """
    mini_num = float(input_list[0])

    for num in input_list[1:]:
        if float(num) <= mini_num:
            mini_num = float(num)

    return mini_num
```

## 注释规范

- **行内注释**：使用中文，解释复杂逻辑
- **块注释**：使用中文，说明代码段的目的
- **用户提示**：使用中文，如 `input('请输入数字')`

## 提交代码规范

### 提交前检查清单

1. **为所有函数添加文档字符串**
   - 每个函数必须有清晰的中文说明
   - 说明函数功能、参数和返回值
   - 如果函数没有文档字符串，提交前必须添加

2. **代码质量检查**
   - 确保代码可以正常运行
   - 检查是否有语法错误
   - 验证逻辑是否正确

3. **提交信息规范**
   - 使用中文编写提交信息
   - 简洁明了地描述功能，如：
     - "查找列表中的最小值"
     - "判断一个数字是否为质数"
     - "反转字符串功能"
   - 避免使用"修复"、"更新"等模糊词汇，直接说明做了什么

4. **代码分析与测试**
   - 在提交代码前，必须先分析代码逻辑和功能
   - 运行代码进行测试，验证功能是否正常
   - 将测试结果反馈给用户
   - 如果发现代码存在问题或可以优化的地方，提出具体的优化方案
   - 询问用户是否需要修改代码
   - 只有在代码测试通过或用户确认后才进行提交

## 代码组织

### 文件结构
```python
# 导入模块（如需要）
import module_name

# 辅助函数
def helper_function():
    """辅助函数说明"""
    pass

# 主要功能函数
def main_function():
    """主要功能说明"""
    pass

# 主程序入口
def main():
    """程序主入口"""
    pass

# 执行主程序
if __name__ == '__main__':
    main()
```

### 建议使用 `if __name__ == '__main__':`
- 将主程序逻辑放在 `main()` 函数中
- 使用 `if __name__ == '__main__':` 保护主程序入口
- 这样可以让代码既能直接运行，也能被导入使用

## 错误处理

- 使用 `try-except` 处理可能的异常
- 给用户友好的中文错误提示
- 验证用户输入的有效性

示例：
```python
def is_valid(input_list):
    """
    验证输入列表是否有效

    参数:
        input_list: 待验证的输入列表

    返回:
        bool: 如果输入有效返回 True，否则返回 False
    """
    try:
        if not input_list:
            return None

        [float(num) for num in input_list]
        return True
    except ValueError:
        return False
```

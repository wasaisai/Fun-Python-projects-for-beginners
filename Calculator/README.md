👉 break 不只可以用在 for 循环里
👉 它可以用在任何循环里：for 和 while 都可以


在 if 语句里，不能直接“退出程序”
👉 你只能：
1️⃣ 退出循环 → 用 break
2️⃣ 退出函数 → 用 return
3️⃣ 立刻终止整个程序 → 用 sys.exit()
用哪个，取决于你现在在什么环境里。
| 场景       | 用什么             |
| -------- | --------------- |
| 在循环里正常退出 | `break` ⭐⭐⭐     |
| 在函数里结束逻辑 | `return` ⭐⭐     |
| 立即杀死程序   | `sys.exit()` ⚠️ |

🤔 问题：为什么不直接写 main()？
    因为以后你的文件可能被别的文件导入，不希望一导入就自动运行
python main.py → ✔️ 程序运行
import main → ❌ 不会自动运行

🧠 先认识 __name__ ✅ __name__ 是什么？
它是 Python 自动提供的一个变量，每个 .py 文件都有一个 __name__，你不用定义，Python 自动给你。

🧠 核心新知识：sys.argv
